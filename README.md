# DP Matrix Rain Effect

A custom "Matrix Rain" visual effect featuring a massive, glowing "dp" logo in the center.

## [ðŸ”´ Live Demo](https://thejohnnyrocket.github.io/dp-matrix-rain/)

## Features

- **Custom Logo**: A large, centered "dp" logo formed by falling characters.
- **Triple Density**: The logo area features 3x the rain density of the background, making it "stick out" prominently.
- **Glow Effect**: Logo characters glow with a bright green aura (`#00ff00`).
- **Uniform Color**: All rain within the logo is pure bright green for a solid, high-contrast look.
- **Italicized Fonts**: Uses italicized Arial Black/Impact for a dynamic, forward-moving aesthetic.
- **Optimized Animation**: Rain speed is tuned for a majestic, hypnotic flow (55ms interval).

## Usage

Simply open `index.html` in any modern web browser.

## Files

- `index.html`: Main entry point.
- `script.js`: Core logic for the matrix effect, logo masking, and animation.
- `style.css`: Basic styling for full-screen canvas.
